# 反向代理及配置

_适用于sudo apt-get install nginx系统级安装_

1. 添加配置

```bash
    sudo vim /etc/nginx/sites-available/reverse-proxy.conf

    server {
        listen 8443;
        server_name your-domain.com;  # 替换为域名或IP

        location / {
            proxy_pass http://127.0.0.1:8001;  # 后端服务地址:端口
            proxy_set_header http_version 1.1;
            proxy_set_header Upgrade $http_upgrade;
            proxy_set_header Connection "upgrade";
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;
        }
    }
```

2. 启用配置
   ```bash
    sudo ln -s /etc/nginx/sites-available/reverse-proxy.conf /etc/nginx/sites-enabled/
    sudo nginx -t                  # 检查语法
    sudo systemctl reload nginx    # 热重载配置
   ```
3. 启动代理服务
   ```bash
    sudo restart nginx
    sudo status nginx
   ```

# 带负载均衡的反向代理和配置

1. 上游配置

```bash
upstream backend {
    server 10.0.0.11:8000 weight=5 max_fails=3 fail_timeout=30s;
    server 10.0.0.12:8000 weight=3 max_fails=3 fail_timeout=30s;
    server 10.0.0.13:8000 weight=1 max_fails=3 fail_timeout=30s;#不同的服务器

    keepalive 32;
}
```
